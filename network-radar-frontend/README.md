# Network Radar

Real-time network monitoring and visualization tool.

## Stack

- React 18 + TypeScript
- Vite
- Framer Motion
- Lucide Icons

## Setup

```bash
npm install
npm run dev
```

## Features

- Real-time network device discovery
- Interactive radar visualization
- Device filtering and search
- Port scanning display
- Responsive design

## Structure

```
src/
├── components/
│   ├── Radar/         # Radar visualization
│   ├── DeviceList/    # Device cards and list
│   └── Layout/        # Dashboard layout
├── hooks/             # Custom hooks
├── types/             # TypeScript definitions
└── utils/             # Helper functions
```
